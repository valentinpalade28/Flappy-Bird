The following is a set of instructions wich will help you play the game.

1. Open the project file with CodeBlocks
2. Run the program with the f9 button
3. Type 2 and press enter to leave the game or type 1 and press enter to start the game
4. Keep pressing enter (recommended) to keep the "bird" (*) in the "sky". The objective of the game being to "fly" it through the spaces in the incoming pipes
5. If the "Game Over" screen appears, either type n and press enter to exit the game or type y and press enter to play again

Good luck and have fun!
